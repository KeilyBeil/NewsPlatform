### 架构说明
- 网络层：HttpUtil（封装 http kit） → ResponseResult（统一响应）
- 数据层：NewsData / NewsTypeModel / NewsModel（本地缓存）
- 业务层：NewsViewModel（单例，封装所有网络调用）
- 常量层：CommonConstant（服务器地址、接口路径、UI 阈值）
- 工具层：Logger / PullDownRefresh / PullUpLoadMore
- UI 层：NewsItem / RefreshLayout / LoadMoreLayout / NewsList / TabBar / Index

### 11:13 追加修复 — 颜色资源格式问题

- **根因**：HarmonyOS 颜色资源必须使用 **8 位 ARGB 十六进制**（如 `#FFFF4040`），不能用 6 位 RGB（`#FF4040`）
- **修复**：`base/element/color.json` 和 `dark/element/color.json` 全部改为 8 位格式
- **额外修复**：`.shadow()` 的 `color` 参数**不支持 `$r('app.color.xxx')`**，需直接传字符串字面量如 `'#1A000000'`
